Loan Calculator Widget

A complete and accessible solution for loan calculations

Widget Features

This loan calculator widget is designed to be easily embedded in any website using an iframe.

  • Responsive and accessible design
  • Real-time validation and calculations
  • Automatic calculations with French amortization
  • Complete form with inline layout
  • Synchronized sliders and numeric inputs
  • Progressive disclosure of contact form
  • Communication with parent page
  • Modern and professional styling

New Features in v2.0

  • English language interface
  • Inline input layout (label left, input right)
  • Dual controls for downpayment and duration
  • Real-time calculations with 5.5% interest rate
  • Progressive contact form reveal
  • Enhanced accessibility and keyboard navigation

Embedding Code

<iframe src="loan-widget/index.html" width="100%" height="560" style="border:0;"></iframe>

Customization

You can adjust the width and height of the iframe according to your needs. The widget automatically adapts to the container.
